# Armor Stand Utils
![A screenshot depicting two Armor Stands - a small one and a big one with arms and no baseplates standing next to each other](showcase.png)

**ASU** allows you to easily modify some Armor Stand properties using Name Tags. Just click the Armor Stand with the correct Name Tag and it will be modified!

## 🌟Features
- **3** modifiable properties
- **Rotation**
- Easy interface using **Name Tags**
- **In-game tutorial**
- **Reset** function

## ✅Usage
Paste this command in the chat and press **ENTER** to see an **in-game tutorial**:
```
/trigger asu_tutorial
```
To use a Name Tag, just **rename** it in an Anvil and **right-click** the Armor Stand with it.

## 📃List of Name Tags
| Name | Effect |
| ---- | ------ |
| **arms** | Gives the Armor Stand **arms**. |
| **noarms** | **Removes** the arms. |
| **noplate** | Removes the Armor Stand's **baseplate**. |
| **plate** | **Returns** the baseplate. |
| **small** | Makes the Armor Stand **small**. |
| **big** | Makes the Armor Stand **big**. |
| **clockwise** | Rotates the Armor Stand 45 degrees **clockwise**. |
| **counterclockwise** | Rotates the Armor Stand 45 degrees **counter-clockwise**. |

## 🔨For Moderators
> **DO NOT DELETE THE DATAPACK BEFORE UNINSTALLING!**

To uninstall, paste this command in chat and press **ENTER**:
```
/function asu:uninstall
```

After that, delete the datapack and run:
```
/reload
```
***

To reset all Armor Stands to their default states, run this:
```
/function asu:reset
```
_(this **does not** reset rotation)_
> **NOTE THAT THIS WILL ALSO AFFECT ARMOR STANDS, WHOSE PROPERTIES WERE CHANGED BY SOURCES, OTHER THAN ASU**
